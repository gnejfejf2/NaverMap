# NMapsMap

## Overview

NAVER에서 SPM 지원 안 해줘서 내가 쓸려고 만든 패키지.

## Requirements

- macOS Ventura 13.3.1 (a)
- Xcode 14.3

## Installation

### Swift Package Manager

```swift
.package(url: "https://github.com/jaemyeong/NMapsMap.git", .upToNextMajor(from: "3.16.2"))
```

## Usage

```swift
import NMapsMap

let mapView = NMFMapView()
```
